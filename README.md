# PADI Ontology Kernel

A semantic governance kernel for multi-agent systems.

## Purpose
Defines:
- RDF ontology (truth model)
- SHACL constraints (structural validity)
- JSON-LD interoperability layer
- SPARQL query interface design
- Epistemic contract for hallucination control

## Principle
Agents do not “know truth”.  
They operate within a constrained semantic graph.
